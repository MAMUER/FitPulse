## ML-классификация состояния

|Состояние|Описание|Рекомендации|
|---|---|---|
|**Восстановление**|Низкий пульс, высокий HRV|Лёгкая активность, растяжка, йога|
|**Базовая выносливость (E1-E2)**|Пульс 65-80% HRmax|Бег/велосипед 45-90 мин|
|**Пороговая выносливость (E3)**|Пульс 80-90% HRmax|Темповые интервалы 30-60 мин|
|**Силовая/HIIT**|Пульс 90-100% HRmax|HIIT, силовые, спринты|
|**Перетренированность**|Низкий HRV + высокий пульс|Отдых 1-3 дня|
|**Заболевание**|Температура > 37.5°C|Прекратить тренировки|

## Генерация тренировок и диеты

Адаптивный план на основе:

- текущего состояния пользователя (ML-классификация)
- цели (похудение, набор мышц, выносливость, гибкость)
- место (дом/зал/бассейн/улица)
- доступные дни и время суток
- подключённые устройства

**Тренировки:** 4–12 недель, упражнения с подходами/повторениями, расписание по дням и времени.

**Диета:** расчёт калорий по формуле Миффлина-Сан Жеора, типы диет (сбалансированная, высокобелковая, для похудения, кето, палео), макронутриенты, приёмы пищи по времени.

## Административные возможности

- Регистрация через **invite-код** (одноразовый)
- Просмотр списка пользователей
- Создание новых invite-кодов через БД функцию `create_invite_code()`
- Мониторинг и аудит действий

## Система invite-кодов

|Код|Роль|Лимит|
|---|---|---|
