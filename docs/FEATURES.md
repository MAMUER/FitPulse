# FitPulse — Возможности платформы

## Вкладки мобильного приложения

|Вкладка|Содержимое|
|---|---|
|**Обзор**|Пульс, SpO₂, сон, давление, AI-анализ, тренировка на сегодня|
|**Профиль**|Возраст, вес, рост, цели, противопоказания, питание, сон|
|**Тренировки**|Активные планы, генерация нового плана, прогресс|

|**Admin**|Панель администратора: пользователи, приглашения, мониторинг|

## ML-классификация состояния

|Состояние|Описание|Рекомендации|
|---|---|---|
|**Восстановление**|Низкий пульс, высокий HRV|Лёгкая активность, растяжка, йога|
|**Базовая выносливость (E1-E2)**|Пульс 65-80% HRmax|Бег/велосипед 45-90 мин|
|**Пороговая выносливость (E3)**|Пульс 80-90% HRmax|Темповые интервалы 30-60 мин|
|**Силовая/HIIT**|Пульс 90-100% HRmax|HIIT, силовые, спринты|
|**Перетренированность**|Низкий HRV + высокий пульс в покое|Отдых 1-3 дня, снизить объём|
|**Заболевание**|Температура > 37.5°C|Прекратить тренировки, обратиться к врачу|

## Генерация тренировок и диеты

Адаптивный план на основе:

- текущего состояния пользователя (классификация)
- цели
- место
- доступные дни и время суток
- подключённые устройства
- особенности здоровья пользователя, влияющие на результат

В том числе учитываются:

- аллергии и заболевания
- инвалидность и ограничения по подвижности
- женский цикл (интеграция с Flo) и связанные с ним физиологические особенности
- динамика веса и состав тела (интеграция с OKOK International)

**Тренировки:** 4–12 недель, упражнения с подходами/повторениями, расписание по дням и времени.

**Диета:** расчёт калорий по формуле Миффлина-Сан Жеора, типы диет (сбалансированная, высокобелковая, для похудения, кето, палео), макронутриенты, приёмы пищи по времени.

## Административные возможности

- Регистрация через **invite-код** (одноразовый)
- Просмотр списка пользователей
- Создание новых invite-кодов через БД функцию `create_invite_code()`
- Мониторинг и аудит действий

## Система invite-кодов

|Код|Роль|Лимит|
|---|---|---|
|`ADMIN-2026-SETUP-ROOT-001`|admin|1|

API: `POST /api/v1/invite/validate` — проверка кода  
API: `POST /api/v1/register/invite` — регистрация через код
