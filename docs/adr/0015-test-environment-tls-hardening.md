# ADR 0015: Инфраструктура тестового окружения — testcontainers-go, graceful TLS, изоляция окружений

## Статус

Принято

## Контекст

Локальный запуск сервисов не поддерживается, deploy только на VPS (`CONTRIBUTING.md`).

## Решение

Интеграционные тесты на `testcontainers-go` — библиотеку, которая поднимает эфемерные контейнеры прямо из тестов. Локальный runtime по-прежнему не поддерживается, но разработчики могут запускать интеграционные тесты локально.

1. **Testcontainers для зависимостей**:
   - PostgreSQL, Valkey, RabbitMQ запускаются автоматически через `testcontainers-go` в каждом тесте.
   - Контейнеры живут ровно столько, сколько живёт тест; после завершения — удаляются.

2. **TLS в тестах**:
   - Self-signed сертификаты генерируются на лету (через `crypto/x509` + `crypto/rand`), без файлов на диске.
   - Gateway в integration-тестах стартует с TLS-сертификатом из памяти.
   - Health-check выполняется по HTTPS с отключённой проверкой CA (`InsecureSkipVerify` или кастомный `tls.Config`).

3. **Graceful TLS skip для unit-тестов**:
   - Gateway стартует без паники при отсутствии TLS-переменных, что позволяет запускать unit-тесты без контейнеров вообще.

4. **Изоляция окружений**:
   - Production/VPS окружение остаётся единственным целевым для runtime.
   - Testcontainers используется исключительно для автоматизированных тестов, а не для имитации production.

## Последствия

- **Плюсы**: интеграционные тесты самодостаточны; разработчики не зависят от внешних CA; сертификаты не попадают в репозиторий.
- **Нейтрально**: требуется Docker Daemon для запуска тестов; self-signed certs не годятся для production.
- **Риски**: при изменении схемы БД нужно синхронизировать миграции в `db/migrations/V1__full_schema.sql` с тестовыми контейнерами.

## Реализация

- `go.mod` — добавлена зависимость `github.com/testcontainers/testcontainers-go`.
- `internal/testcontainers/` — helpers для поднятия PostgreSQL, Valkey, RabbitMQ контейнеров.
- Интеграционные тесты (`*_integration_test.go`) используют `testcontainers-go`.
- `cmd/gateway/main.go` — логика graceful TLS fallback реализована.
